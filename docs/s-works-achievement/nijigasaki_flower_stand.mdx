[スクールアイドルプロジェクト「ラブライブ！」シリーズ](https://lovelive-anime.jp/) の [虹ヶ咲学園スクールアイドル同好会 5th ライブ](https://www.lovelive-anime.jp/nijigasaki/sp_5thlive.php) に向けた、ユーザー参加イベント用の Web アプリを開発しました。

祝花をスマートフォンや PC 上で作成して、5th ライブに参加するメンバーを応援するイベントです。
ユーザーは花、メンバー、小物など様々なパーツを自由に組み合わせて、思い思いの祝花を作成することが出来ます。
