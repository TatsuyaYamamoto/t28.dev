[イキヅライブ! LOVELIVE! BLUEBIRD](https://www.lovelive-anime.jp/lovehigh/) が約１年間にわたって投稿していた、
いきづらい部！メンバーのポストを閲覧できるアーカイブページを開発しました。

大ボリュームの投稿から "捗る" 投稿を見つけられるように、キーワード・日付・メンバーで検索できます。検索の速度も売りです。
元の投稿が X にあったものなので、アーカイブページの投稿もそれに寄せたデザインにして、オリジナルの雰囲気を楽しめるようにしました。
