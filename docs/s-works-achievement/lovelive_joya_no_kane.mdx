[スクールアイドルプロジェクト「ラブライブ！」シリーズ](https://lovelive-anime.jp/) の 2021 年の年末企画用 Web アプリを開発しました。

アプリではシリーズ内の 4 作品における主人公を操作して除夜の鐘を鳴らすことが出来ます。
ユーザーは鐘を 108 回鳴らして、2022 年の抱負や願い事をツイートしながら [#ラブライブ大晦日 2021](https://twitter.com/hashtag/%E3%83%A9%E3%83%96%E3%83%A9%E3%82%A4%E3%83%96%E5%A4%A7%E6%99%A6%E6%97%A52021?src=hashtag_click) を過ごす、という企画です。

各ユーザーの鐘を鳴らした回数は集計され、タイトル画面でリアルタイムの合計を見ることが出来ます。
さらに、操作によって鐘の音(強弱)が変わったり、ゴール(108 回)以外の回数(50 回、80 回、1080 回)でカットインイラストを表示されることで、瞬間的な面白さとゴールに向けた継続的な興味喚起の仕組みを作りました。

その結果、企画用ハッシュタグ( [#ラブライブ大晦日 2021](https://twitter.com/hashtag/%E3%83%A9%E3%83%96%E3%83%A9%E3%82%A4%E3%83%96%E5%A4%A7%E6%99%A6%E6%97%A52021?src=hashtag_click) )は 年末の Twitter トレンドの上位をキープすることが出来ました。

最終的に鐘は 7287503 回鳴らされました。
