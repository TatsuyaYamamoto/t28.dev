[スクールアイドルプロジェクト「ラブライブ！」シリーズ](https://lovelive-anime.jp/) の [虹ヶ咲学園スクールアイドル同好会](https://www.lovelive-anime.jp/nijigasaki/) 4th ライブに向けた、応援メッセージ投稿企画用の Web アプリを開発しました。

ライブに出演する 12 人のキャラクターに向けてデジタルメッセージカードを送る企画です。
ユーザーはカード台紙とフォントを選び、メッセージを入力することで、メッセージカードを作成することが出来ます。

Twitter でメッセージを共有すると、メッセージカードの画像もツイートに表示されます。
企画用ハッシュタグ ( [#みんなで作る LLL](https://twitter.com/search?q=%23%E3%81%BF%E3%82%93%E3%81%AA%E3%81%A7%E4%BD%9C%E3%82%8BLLL%20until%3A2022-01-13&src=typed_query&f=live) )からはメッセージ(文字)だけでなく、キャラクターのために選んだカードやフォントも併せて表示されるため、みんなで作っている感、みんなでワクワク待っている感、が分かりやすくなっています。

送られたメッセージカードはライブの演出にも使用されました。
