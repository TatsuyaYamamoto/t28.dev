[スクールアイドルプロジェクト「ラブライブ！」シリーズ](https://lovelive-anime.jp/) 、 [虹ヶ咲学園スクールアイドル同好会](https://www.lovelive-anime.jp/nijigasaki/) から、
LoveLive!Days ラブライブ！総合マガジンで連載されていた『[レジェンド オブ ニジガク ～虹の守護者～](http://gs.dengeki.com/news/150261/) 』をまとめた[ファンブック](https://www.kadokawa.co.jp/product/322202000957/) が発売されます。

今回、ノベルの発売にあわせた企画用の Web アプリを開発しました。

ユーザーはダイスを回して、おすすめのキャラ・職業を占うことが出来ます。
本ノベルは "TRPG (テーブルトーク RPG) 風ノベル" なので、カードをめくったり、ダイスを3次元的に転がす操作でアナログゲームっぽさを出してみました。

推し的には "魔法使い" になりたいものですが、お仕事柄、私には "紋章師" が性に合っていそうです。
